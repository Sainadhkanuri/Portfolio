IOT based Smart Biometric E-Attendance System

Biometric Attendance systems are commonly used systems to mark the presence in offices.The IOT Based Smart Biometric E-Attendance system in any organization or Industry can get the efficient attendance with accurate date and time. For this we have interfaced fingerprint sensor with NodeMCU, OLED Display and Buzzer. NodeMCU is an open-source LUA script based firmware developed for the ESP8266 wifi chip With the help of Finger Print Scanner R307, we will store the finger prints of all the users and once they are stored, the Finger Print Scanner will compare the present finger print on the scanner and previously stored finger prints. Using this system one can back-up the attendance of the users at any time.
